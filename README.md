## II. ĐỊNH NGHĨA NỀN TẢNG & KIẾN TRÚC

Toilabap.com là nền tảng algo trading mã nguồn mở, kiến trúc Blankly Framework, bridge thông minh giữa logic chiến lược và mọi sàn giao dịch.

**Triết lý:** "Write your strategy logic once. Run it on any exchange, any asset class, without changing a single line of trading logic."

**Stack:**
```
┌─────────────────────────────────────────────────────────┐
│                    YOUR STRATEGY LOGIC                  │
│         (price_event → entry/exit/sizing/risk)          │
├─────────────────────────────────────────────────────────┤
│              TOILABAP.COM FRAMEWORK LAYER               │
│   Backtest Engine │ Paper Trade │ Sandbox │ Live Deploy  │
├─────────────────────────────────────────────────────────┤
│              EXCHANGE ABSTRACTION LAYER                 │
│  Alpaca │ Interactive Brokers │ Binance │ Coinbase │ ... │
├─────────────────────────────────────────────────────────┤
│                 ASSET CLASS COVERAGE                    │
│   Stocks (VN/US) │ Crypto │ Forex │ Futures │ Hàng Hóa  │
└─────────────────────────────────────────────────────────┘
```

**Tích hợp:** Python, NodeJS, Excel, Google Spreadsheet, Amibroker, MetaTrader, mọi broker lớn.
